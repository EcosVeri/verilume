# 🛣️ Verilume Roadmap

> Answers powered by local retrieval, trusted AI generation, and transparent sources.

This roadmap outlines the planned evolution of Verilume. Features may change as
the community grows.

---

# Version 1.0 — Foundation ✅

Core retrieval-augmented generation platform.

- [x] Local document ingestion
- [x] PDF support
- [x] DOCX support
- [x] Markdown support
- [x] TXT support
- [x] CSV support
- [x] Chroma vector database
- [x] Hybrid local retrieval
- [x] Tavily web search
- [x] Hugging Face inference
- [x] Conversation memory
- [x] Source transparency
- [x] Confidence estimation
- [x] Export conversation to Markdown
- [x] Export conversation to PDF

---

# Long-Term Vision

Verilume aims to become an open-source AI workspace where users can:

- Chat with documents
- Search the web
- Analyze images
- Understand scanned files
- Review research papers
- Generate reports
- Build AI workflows
- Run everything locally whenever possible
- Install Verilume as a polished desktop app on macOS
- Install Verilume as a Python package from PyPI or GitHub
- Use a Windows executable if the Windows build pipeline proves reliable
