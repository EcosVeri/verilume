# FAQ

## What is Verilume?

Verilume is a local-first AI research assistant for documents, AI knowledge, and optional web evidence.

## Does Verilume keep documents local?

Uploaded documents and the Chroma database are stored locally by default under `~/.verilume`.

## Does web search have to be enabled?

No. Web search is optional. Users can run local-only workflows when they want answers restricted to indexed files.

## Which model backends are supported?

Verilume supports Hugging Face and Ollama generation backends.

## What citations does Verilume use?

Local citations use `[S1]`, `[S2]`, and `[S3]`. Web citations use `[W1]`, `[W2]`, and `[W3]`.

## Can Verilume read scanned documents?

Yes. Verilume supports OCR for scanned PDF pages and image uploads.

## Can chats be exported?

Yes. Chats can be exported to Markdown and PDF.
